Detailed Description

Egy manualisabb idomero, ami a rendszeridot hasznalja. A manualis resze abbol adodik, hogy amikor kell konkret ido adat, akkor le kell (vagy akkor le lesz) frissitve, igy nem kell kulon szalon futnia.
